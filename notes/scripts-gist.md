# Scripts Gist — All Python Fix Commands

> **How this was generated**: Each `plan.md` contains a `<!-- SCRIPTS_HOOK_START ... -->` block listing script todos.  
> This file is the extracted summary — run these commands in order once scripts are generated.
>
> **To regenerate all scripts in one pass**, run:
> ```bash
> python scripts/generate_all_fix_scripts.py
> ```
> That script reads all `plan.md` files, parses the `SCRIPTS_HOOK` blocks, and writes every listed Python script under `scripts/`.

## Script Locations

All scripts live under `scripts/` at the repo root. They are generated by:

```bash
python scripts/generate_all_fix_scripts.py
```

That script scans all `notes/**/plan.md` files, finds `<!-- SCRIPTS_HOOK_START -->` blocks, extracts each row from the scripts table, and writes the corresponding Python file.
